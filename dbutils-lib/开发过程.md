# 数据库记录用类型表示

一条记录有多少种表示方法

- 实体类的一个对象（bean）
- 数组的一个对象（object[])
- 一个map对象
- 等等

要表示多条记录，就需要用集合，而集合，现在主要是用

- Map
- List
- 等等

基于上面的分析，所以要表示数据库的记录常见的就是3*2这6种情况

- List<实体>
- List<Object[]>
- List<Map>
- Map<Map>>
- Map<实体>
- Map<Object[]>
